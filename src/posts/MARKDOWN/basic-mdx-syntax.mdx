## Heading

Heading is commonly used to define something important or give attention to reader 
We can use it by adding hash(#), and everytime hash(#) increases smaller the size of text it is

```markdown
hash from 1 ~ 6
# 1 hash Biggest
## 2 hash
### 3 hash
#### 4 hash
##### 5 hash
###### 6 hash Smallest
####### 7 doesn't work
```
 
#### Heading Result

# 1 hash Biggest

## 2 hash

### 3 hash

#### 4 hash

##### 5 hash

###### 6 hash Smallest

## Ordered List

Ordered list is define by number with a dot

Even though we define list unorderly, it does not order by itself<br/>
please make sure that the number is ordered

```markdown
1. one
2. two
3. three

1. one
3. three
2. two
```

#### Ordered List Result

1. one
2. two
3. three

## Unordered List

Unordered List is define by -

```markdown
- parent
    - child
        - descendent
```

#### Unordered List Result

- parent
  - child
    - descendent

## Check List

Check List is define by -`[ ] for uncheck [x] for check`

```markdown
* [x] finish
* [ ] todo

```

#### Check List Result

* [x] finish
* [ ] todo

## Delimeter

Delimeter is define by `<hr/>` `---`

```markdown
between any contents

---

<hr/>
```

#### Delimeter Result

contents

<hr />

---

## URL Link / Email Link

Link is define by `[name](link)`

```markdown
URL Link
[Google](https://google.com)

```

#### Link Result

Link: [Google](https://google.com)<br/>

## Text Type

Text Type of Strong, em, strike

```markdown

I am _Seogki Hong_ (emphasis)

I am **Seogki Hong** (strong)

I am ~Seogki Hong~ (strike)
I am ~~Seogki Hong~~ (strike)

```

#### Text Type Result

I am _Seogki Hong_ (emphasis)

I am **Seogki Hong** (strong)

I am ~Seogki Hong~ (strike)

I am ~~Seogki Hong~~ (strike)

## Table

Table is define by combination of `|`, `:`, `-`

```markdown
| a | b  |  c |  d  |
| - | :- | -: | :-: | 
| 1 | 2  |  3 |  4  |
```

| a | b  |  c |  d  |
| - | :- | -: | :-: |
| 1 | 2  |  3 |  4  |
